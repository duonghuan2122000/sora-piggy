Hướng dẫn ngắn về i18n trong Sora

- Thêm key mới:
  - Mở file localization (ví dụ src/renderer/src/locales/vi.json)
  - Thêm key theo cấu trúc phân cấp, ví dụ:
    {
    "transactions": {
    "title": "Giao dịch",
    "columns": {
    "date": "Ngày",
    "amount": "Số tiền"
    }
    }
    }

- Sử dụng trong component:
  - import { useI18n } from 'vue-i18n'
  - const { t } = useI18n()
  - Sử dụng t('transactions.title') trong template hoặc script

- Lưu ý:
  - Nếu cần mặc định cho fallback, có thể dùng t('key') || 'Default'
  - Sau thay đổi file ngôn ngữ, restart dev server nếu không thấy cập nhật
