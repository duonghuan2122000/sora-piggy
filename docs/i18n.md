# i18n Guidelines

Vị trí file locales: src/locales/vi.json (và src/locales/en.json nếu có)

Quy ước key: domain.feature.component.element (ví dụ transactions.list.title)

Usage:

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n';
const { t } = useI18n();
</script>

<template>
  <h1>{{ t('transactions.title') }}</h1>
</template>
```

When adding new keys, update src/locales/vi.json and add translation. Avoid using raw strings in templates.
